# Проекттің архитектурасы және жұмыс істеу принципі

Бұл құжат сенің сайтыңның қалай құрылғанын, «қозғалтқышы» (engine) не екенін және әрбір файлдың қызметін түсіндіруге арналған.

## 1. Негізгі «Қозғалтқыш» (Framework)
Сенің сайтың **Flask** (Python) негізінде жасалған. 
*   **Flask** — бұл веб-фреймворк. Ол браузерден келген сұраныстарды қабылдап, оларға жауап қайтаруға жауапты.
*   **Деректер базасы:** SQLite (файл түріндегі база).
*   **Шаблондар:** Jinja2 (HTML файлдарының ішіне Python логикасын жазуға мүмкіндік береді).

---

## 2. Файлдардың құрылымы (File Structure)

### Түбірлік каталог (Root):
*   `run.py` немесе `app.py`: Сайтты іске қосатын негізгі файл. Проекттің «кілті» деуге болады.
*   `config.py`: Сайттың баптаулары (базаның жолы, құпия кілттер, сурет жүктеу шектеулері).
*   `requirements.txt`: Сайт жұмыс істеуі үшін қажетті барлық Python кітапханаларының тізімі.
*   `news_kz.db`: Барлық жаңалықтар, санаттар және админдер сақталатын деректер базасы.
*   `rss_parser.py`: Автоматты түрде басқа сайттардан жаңалықтарды жинайтын «робот» (скрипт).

### `app/` қалтасы (Негізгі логика):
*   `__init__.py`: Проектті бір бүтін етіп біріктіреді, базаны және басқа компоненттерді баптайды.
*   `models.py`: Мәліметтердің құрылымы. Мысалы: Жаңалықтың тақырыбы, мәтіні, уақыты қандай болуы керек екенін анықтайды.
*   `routes.py`: Пайдаланушылар көретін беттердің логикасы (Басты бет, жаңалық оқу беті).
*   `admin.py`: Админ панельдің логикасы (жаңалық қосу, өшіру, баптаулар).
*   `translations.py`: Сайтты үш тілге (қазақ, орыс, ағылшын) аударуға арналған сөздік.

## 3. Сайт қалай жұмыс істейді? (Workflow)

Сайттың жұмыс істеу циклі 5 қадамнан тұрады:

1.  **Сұраныс (Request):** Пайдаланушы браузерге `yoursite.com/news/1` деп жазады.
2.  **Маршрут (Route):** Flask бұл мекенжайды `routes.py` ішінен іздейді.
3.  **Логика (Logic):** `routes.py` ішіндегі функция базадан (SQLite) №1 жаңалықты тауып алуды сұрайды.
4.  **Деректер (Data):** `models.py` арқылы базадан ақпарат алынады.
5.  **Көрініс (Template):** Алынған ақпарат `app/templates/news.html` файлына жіберіледі. Бұл файл деректерді әдемі HTML форматына салып, пайдаланушының браузеріне қайтарады.

---

## 4. RSS Парсер қалай жұмыс істейді?

Бұл сенің сайтыңның ең маңызды бөлігі. 
*   Скрипт (`rss_parser.py`) белгіленген уақыт сайын (мысалы, әр 30 минутта) басқа сайттардың RSS таспасын тексереді.
*   Жаңа жаңалық тапса, оны автоматты түрде аударып, базаға сақтайды.
*   Сондықтан сенің сайтыңда әрдайым жаңа ақпарат болады.

---

## 5. Технологиялық стек (Summary)
*   **Backend:** Python 3 + Flask
*   **Frontend:** HTML5, CSS3, JavaScript (Glassmorphism design)
*   **Database:** SQLite (SQLAlchemy ORM)
*   **Auth:** Flask-Login
